SillyTavern 云酒馆使用指南

重要说明：夜鹭机安装的是 SillyTavern 原生版本，没有预装任何插件或预设。你需要自己配置 AI 接口和导入角色卡。

一、首次访问与登录

1 打开云酒馆

在夜鹭机仪表盘中点击云酒馆卡片的「打开」按钮

或手动访问 http://{{ server_ip }}:{{ tavern_port }}

2 选择用户登录

打开后你会看到一个用户选择界面。你必须选择安装时设置的那个用户名，然后输入密码登录。

如果忘记了用户名和密码，可以在夜鹭机仪表盘的「云酒馆登录信息」卡片中查看。

不要选择 Default User（默认用户）！
默认用户没有密码，任何人知道你的地址和端口就能直接进入使用，会白白消耗你的 API 额度。
请务必选择你自己设置的用户名，输入密码后登录。

二、配置 AI 接口（最重要的第一步）

1 打开 API 设置

登录后，点击顶部工具栏的插头图标（API 连接按钮），打开 API 设置面板。

2 切换到聊天补全模式

在 API 设置面板顶部，你会看到一个下拉菜单（默认可能是 Text Completion）。请将其切换为 Chat Completion。

Chat Completion 是目前最主流的 API 格式，OpenAI、DeepSeek、Claude 中转、Gemini 等都使用这个格式。

必须先切换到 Chat Completion！这是新手最容易忽略的一步。如果不切换，你填了 API Key 也可能无法正常使用。
3 选择 AI 来源

切换到 Chat Completion 后，在下方选择你的 AI 来源：
- OpenAI：适用于 OpenAI 官方、DeepSeek、国内中转等（最通用）
- Claude：适用于 Anthropic Claude 官方 API
- Custom (OpenAI-compatible)：适用于其他兼容 OpenAI 格式的 API

4 填写 API 信息

以 OpenAI 兼容格式为例（适用于大多数中转和 DeepSeek）：
- API Key：填入你的密钥
- API URL：填入 API 地址（如 DeepSeek 是 https://api.deepseek.com）
- 填完后点击「Connect」测试连接

没有 API Key？你需要先去 AI 服务商注册并获取。常用的有 DeepSeek（便宜好用）、OpenAI、Claude 等。API 是按使用量计费的。

三、导入角色卡

1 获取角色卡

角色卡是 AI 角色的人设文件。推荐获取渠道：
- Chub.ai（最大的角色卡分享平台）
- 各种社区群分享
- 自己在 SillyTavern 里创建

2 导入角色

点击左侧角色列表上方的「导入」按钮，选择角色卡文件（PNG/JSON 格式）。也可以直接把文件拖进页面。

四、开始对话

1 选择角色并开始聊天

在角色列表中点击想对话的角色，在底部输入框输入消息，按回车发送。

快捷键：Enter 发送 / Ctrl+Enter 换行 / 上箭头 编辑上一条

五、手机访问

手机浏览器直接输入 http://{{ server_ip }}:{{ tavern_port }} 即可访问。

添加到主屏幕（像 App 一样用）：
- iPhone：Safari 打开 → 点分享 → 添加到主屏幕
- Android：Chrome 打开 → 右上角菜单 → 添加到主屏幕

六、常见问题

Q: 打不开酒馆页面？

1. 确认用的是 http:// 不是 https://
2. 检查端口是否在云服务商控制台放行了
3. 在夜鹭机仪表盘检查云酒馆服务是否运行中

Q: 发消息没有回复？

1. 确认已切换到 Chat Completion 模式
2. 检查 API Key 是否正确填写，点「Connect」测试
3. 检查 API 余额是否充足

Q: 多人可以同时使用吗？

可以。每个用户有独立的角色卡、聊天记录和设置，互不干扰。但 API 费用共享，由你承担。
