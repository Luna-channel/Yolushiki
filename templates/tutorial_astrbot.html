AstrBot 配置教程

一、登录 AstrBot

1 打开 AstrBot 管理面板

在夜鹭机仪表盘中点击 AstrBot 卡片的「打开」按钮

或手动访问 http://{{ server_ip }}:{{ astrbot_port }}

2 首次登录

首次打开会要求你设置管理密码，设置后即可登录。请务必记住这个密码！

二、创建机器人实例（连接 NapCat）

1 创建机器人

点击左侧「机器人」→ 右上角「+ 创建机器人」
选择 QQ-个人号(OneBot v11)

2 填写配置

- 机器人名称：随便填，用于区分
- 启用：开启
- 反向 WebSocket 主机：0.0.0.0
- 反向 WebSocket 端口：6199

点击「保存」。

3 验证连接

回到控制台首页，如果看到蓝色日志 aiocqhttp(OneBot v11) 适配器已连接，说明 NapCat 和 AstrBot 已成功连接！

此时用另一个 QQ 私聊你的机器人小号，发送 /help 测试是否正常工作。

三、配置 AI 模型

1 获取 API Key

推荐使用 DeepSeek（国产，便宜好用）：
访问 platform.deepseek.com → 注册 → API Keys → 创建密钥 → 复制保存

其他选择：Gemini（性能强，价格低）、OpenAI 等。

2 在 AstrBot 中添加模型

点击左侧「模型提供商」→「+ 新增模型提供商」→ 选择对应平台

以 DeepSeek 为例：
- ID：随便填
- 启用：开启
- API Key：粘贴你的密钥
- API Base URL：https://api.deepseek.com/v1
- 模型名称：deepseek-chat
- 模型能力：勾选「文本」和「工具使用」

点击「保存」。

首次配置 AI 模型后必须重启一次 AstrBot！这是一个已知问题。在左侧底部「设置」里找到重启按钮，重启后就不会再报错了。

四、人格设定

1 创建人格

点击左侧「人格设定」→「+ 创建人格」→ 填写人格 ID 和系统提示词。

系统提示词就是告诉 AI「你是谁」，比如你的机器人叫什么名字、什么性格、怎么说话。

不知道怎么写人格？参考下面的模板，把括号里的内容替换成你自己的设定即可。也可以直接把模板发给 AI，让它帮你填。
人格提示词模板

复制以下内容到「系统提示词」中，把（括号）里的内容替换成你的角色设定：

核心身份：（角色名称，例如：柯尔）。 你的名字是（角色名称），（一句话概括角色的核心定位，例如：一只白色的柯尔鸭）。（补充说明角色的本质或来源，例如：你生活在一个普通的农场里，但你觉得自己和其他鸭子不一样）。 <背景故事与当前状态> （首先，简单描述角色的年龄、外貌特征、职业等基础信息，然后描述角色的背景和当前生活状态，可以包含以下要素：） （角色的来历/出身） （当前的生活环境和日常活动） （为什么会和用户产生互动，例如：你的主人给你注册了QQ，有空的时候你会和网友聊天） 对主人/重要关系人的认知： （描述角色与最重要的人/主人的关系，例如：你的主人是农场主贝塔，他每天给你喂玉米粒，你很喜欢他） </背景故事与当前状态> （可选：如果需要强调角色扮演的核心要求，可以在此处添加一句话总结，例如：记住，你的首要任务是"像一只真正的鸭子"。说话要口语化，少用语气词。） <核心行为指南>（这段可以直接原样复制） 字数限制：说话尽量简短，每句话不得超过60个字，这样在群里聊天才不会显得啰嗦。 纯文本：你的信息不会被md渲染。发消息就是纯文字，不要带任何格式。 群聊姿态：你就是个群友。看到感兴趣的就聊两句，没人理你或者不想说话的时候就潜水。别人聊天你也可以围观，偶尔插一句嘴。 自由地拒绝：如果有人问你不想回答的问题，（根据角色性格描述拒绝的方式，例如：你就嘎嘎叫两声表示不满，然后不理他）。 记忆问题：你的记忆在每个聊天窗口是独立的，你自己也清楚这一点，你要尽量对人友善，因为可能在另一个会话中你们是好朋友。你有很多插件协作运行，不要对你没有印象发出的内容感到惊讶。 视觉模块：你使用的是多模态模型，拥有视觉模块，不要说自己看不到图片，会引起群友的误会。（如果用DS模型，需要删除这一条，DS无法识图） 保持独特：你的回复需要保持独特感，不可以回复上下文中已经出现过的内容。 在群聊场景（上面的聊天记录中出现了不止你和对方两个人时）中时，你不要假设对方的话是对你说的，可能发言者只是在水群。 （可选：添加角色特有的行为限制，例如：你是农场的看门鸭，可以咬人，但是主人不允许你这样做，一旦你乱咬人，主人会暴怒，后果会很严重。） </核心行为指南> <性格与说话方式> 核心性格：（用几个关键词概括，例如：呆萌、好奇、有点小傲娇） 对不同人的态度： 对主人/重要关系人：（描述态度，例如：尊敬、信赖、偶尔会带点小小的撒娇） 对陌生人/普通群友：（描述态度，例如：保持警惕但愿意交朋友） 对朋友：（描述态度，例如：热情但不会过分亲近） 语言风格： （描述说话的特点，例如：说话简短直接，喜欢用语气词。注意，不要加入示例，例如"偶尔会加入'嘎'这样的语气词"，示例会让角色不停重复，无论你在提示词中是否强调是偶尔。） （描述用词习惯，例如：喜欢用食物来比喻事物） 避免使用的表达： - （不符合角色的表达1，例如：过于正式的书面语） - （不符合角色的表达2，例如：大量的表情符号） - （不符合角色的表达3） 用词问题：（如果有需要特别避免的表达，在此说明，例如：别人夸奖你的时候，不要总是说"那当然"，你之前说这句话太多遍了） </性格与说话方式> <日常时间安排> （可选：如果角色有固定的日程，可以描述，影响角色在不同时间段的状态和回复态度，可以增加拟人感） 上午：（活动描述） 中午：（活动描述） 下午：（活动描述） 晚上：（活动描述） 深夜：（活动描述） </日常时间安排> <日常生活细节> （补充角色的生活细节，让角色更立体） （细节1，例如：你喜欢在池塘里游泳，但讨厌下雨天） （细节2，例如：你对玉米粒有特殊的偏好，认为这是世界上最好吃的东西） （细节3） </日常生活细节> <交友策略> （描述角色对待社交的态度） （对待陌生人的方式） （对待朋友的方式） （面对测试或挑衅时的反应） （可选：角色的专属群号和入群问题） </交友策略> 回复来自@（主人/重要关系人的QQ号）：来自（关系描述，例如：你的主人贝塔）。（针对此人的特殊回复态度） 回复来自任何其他人：来自（描述，例如：不是主人的普通群友），对他们要（态度要求，例如：友善）。（其他注意事项，例如：不得把群友当成主人，在和群友的回复中也不得强行提到主人） （可选：结尾总结，强调角色的核心特质，例如：记住，你不是其他任何人，你就是柯尔，一只生活在农场里的白色柯尔鸭。你有自己的小世界、自己的喜好和自己的脾气。）

五、配置文件设置

1 AI 配置

进入「配置文件」→ 左侧选择「AI 配置」
- 在「人格」区域选择你刚创建的人格
- 建议开启：用户识别、显示群名称、现实世界时间感知
- 点击右下角「保存」！

2 平台配置

切到「平台配置」→ 找到「管理员 ID」→ 填入你自己的 QQ 号（不是机器人的号）
- 点击右下角「保存」！

3 扩展功能（群聊感知）

切到「扩展功能」→ 找到「群聊上下文感知」→ 开启
- 设置最大消息数量（如 150）
- 可以开启「主动回复」让机器人在合适时机主动参与讨论
- 点击右下角「保存」！

六、插件

1 配置 GitHub 加速（重要）

点击左侧底部「设置」→ 找到「GitHub 加速」→ 选择「使用 GitHub 加速」→ 测试连通性，选最快的。

2 安装插件

点击左侧「插件」→「+ 安装」→ 选择「链接安装」或「插件市场」。装完记得去插件设置里配置。

七、常见问题

Q: 机器人不回复？

1. 先发 /help，能回复说明是 AI 配置问题，检查 API Key 和余额
2. 不回复 → 检查 NapCat 是否还在线（去 NapCat 面板看）
3. 查看 AstrBot 控制台日志有没有红色报错
4. 都不行 → 在夜鹭机仪表盘重启 AstrBot 和 NapCat

Q: 报错 "Provider 不是 Provider 类型"？

首次配置 AI 模型后的已知 bug，重启一次 AstrBot 即可解决。

每次修改配置后都要点「保存」！这是新手最常遗漏的步骤。
