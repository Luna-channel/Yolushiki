# 🌙 夜鹭机 —— AstrBot + SillyTavern 一键部署工具

夜鹭机是一个面向纯小白用户的一键部署工具，帮助你在云服务器上快速搭建 **AstrBot**（QQ 机器人框架）和 **SillyTavern**（云酒馆）。

全程 Web 图形界面操作，无需任何 Linux 基础。

## 部署完成后你将获得

| 服务 | 说明 | 默认端口 |
|------|------|----------|
| **SillyTavern** | 云端酒馆，支持多种 AI 模型 | 自定义（推荐 10000+） |
| **AstrBot** | QQ 机器人框架 | 6185 |
| **NapCat** | QQ 协议端（配合 AstrBot 使用） | 6099 |
| **夜鹭机管理面板** | 服务管理/监控/教程 | 9999 |

## 📦 安装前准备

### 1. 你需要什么

- **一台云服务器**（2 核 2G 内存起步，推荐 2 核 4G）
  - 系统选择：推荐 **Ubuntu 22.04** 或 **Debian 12**，也支持 CentOS/RHEL/Fedora/Arch/openSUSE
  - 需要有**公网 IP**
- **一个能连接服务器的工具**（下面会教你怎么用）

### 2. 推荐的服务器连接工具

你需要一个 SSH 工具来连接你的服务器。以下任选一个：

| 工具 | 平台 | 推荐度 | 说明 |
|------|------|--------|------|
| **1Panel** | 网页版 | ⭐⭐⭐⭐⭐ | 自带文件管理器 + 终端，最适合小白 |
| **宝塔面板** | 网页版 | ⭐⭐⭐⭐⭐ | 国内最流行的服务器管理面板，功能丰富 |
| **Termius** | Win/Mac/手机 | ⭐⭐⭐⭐ | 好看好用，支持手机操作 |
| **FinalShell** | Win/Mac | ⭐⭐⭐ | 国产工具，功能全面 |

> 💡 **强烈推荐使用可视化面板**：1Panel 或宝塔面板都可以。买服务器时很多厂商（腾讯云、阿里云等）可以直接预装，这样你就有网页版的文件管理器和终端，操作最方便。

### 3. 安全组/防火墙端口开放

在开始安装前，你需要在服务器提供商的控制台（不是服务器里面）开放以下端口：

| 端口 | 用途 |
|------|------|
| 9999 | 夜鹭机管理面板 |
| 6185 | AstrBot |
| 6099 | NapCat |
| 你自定义的端口 | SillyTavern（安装时会让你填） |

**如何开放端口**（以腾讯云为例）：
1. 登录腾讯云控制台
2. 找到你的服务器 → 安全组
3. 添加入站规则，协议选 TCP，端口填上面的端口号
4. 保存

> ⚠️ 每个云服务商的操作界面不同，但都叫"安全组"或"防火墙"。如果找不到，搜索"你的云服务商名称 + 开放端口"。

## 🚀 安装教程

### 第一步：上传安装包到服务器

**方法 A：使用 1Panel 文件管理器（最简单）**

1. 在浏览器打开 1Panel 管理页面
2. 左侧菜单点击「文件」
3. 导航到 `/opt` 目录
4. 点击「上传」，把**整个 `yolushiki` 文件夹**拖进去（不是单个文件，是整个文件夹）
5. 确认上传完成，目录结构应该是：
   ```
   /opt/yolushiki/
   ├── app.py
   ├── install.sh
   ├── static/
   │   └── logo.png
   └── templates/
       ├── index.html
       ├── login.html
       ├── tutorial_napcat.html
       ├── tutorial_astrbot.html
       └── tutorial_tavern.html
   ```

**方法 B：使用 SFTP 工具（Termius/FinalShell）**

1. 打开你的 SSH 工具，连接服务器
2. 使用文件传输功能，把 `yolushiki` 文件夹上传到 `/opt/` 目录下

> ⚠️ **注意**：必须上传**整个文件夹**，不是单个文件。文件夹名必须是 `yolushiki`，路径必须是 `/opt/yolushiki/`。缺少任何文件都会导致安装失败并提示你重新上传。

### 第二步：运行安装脚本

1. 打开终端（1Panel 的终端、Termius 或 FinalShell 的 SSH 连接）

2. **复制粘贴**以下命令并按回车：
   ```bash
   bash /opt/yolushiki/install.sh
   ```

3. 你会看到类似这样的输出：
   ```
   ╔═══════════════════════════════════════════════════════╗
   ║              🌙  夜  鹭  机  🌙                      ║
   ║         AstrBot + SillyTavern 一键部署                ║
   ╚═══════════════════════════════════════════════════════╝

   [1/5] 检测系统环境...
         ✓ 系统: Ubuntu 22.04.3 LTS
         ✓ 内存: 3948MB
   [2/5] 检查依赖环境...
   ```

4. 脚本会自动安装所需依赖（Python3、pip3、Flask），**等待 1-2 分钟**

5. 当你看到以下信息时，说明安装向导已启动：
   ```
   [5/5] 启动安装向导...
         ✓ 管理面板已启动
   ╔═══════════════════════════════════════════════════════╗
   ║  请在浏览器中打开:                                    ║
   ║  http://你的服务器IP:9999                             ║
   ╚═══════════════════════════════════════════════════════╝
   ```

> 💡 **记住这里显示的 IP 地址和端口**，下一步要用。

### 第三步：浏览器操作 —— 设置 Token

1. 在浏览器地址栏输入：`http://你的服务器IP:9999`

2. **首次打开**会弹出「设置访问令牌」窗口：
   - 输入你想设置的 Token（相当于管理面板的密码）
   - 再次输入确认
   - 点击确定

   > ⚠️ **请记住这个 Token！** 以后每次登录管理面板都需要它。

3. 设置完成后自动跳转到安装向导页面

### 第四步：配置安装选项

安装向导页面上，你需要配置以下内容：

**端口配置**：
- AstrBot 和 NapCat 端口是固定的（6185、6099）
- **SillyTavern 端口**：必须自定义，建议填 **10000-65535** 之间的数字
  - 例如：`18888`、`23333`、`45678`
  - 不要用常见端口（如 80、443、8080）
  - **这个端口号记住，之后访问酒馆要用**

**安装选项**：
- ☑️ 安装 SillyTavern —— 勾选
- ☑️ 安装 AstrBot（含 NapCat）—— 勾选
- ☑️ 预装 AstrBot 插件 —— 建议勾选

确认配置后，点击 **「开始安装」**。

### 第五步：等待安装完成

安装过程大约需要 **5-20 分钟**，取决于你服务器的网络速度。

**安装过程中你会看到**：
- 进度条和百分比
- 实时安装日志（绿色文字滚动）
- 镜像加速选项面板

**安装速度很慢怎么办？**

如果日志中显示下载速度只有几十 KB/s，或者长时间没有进度变化：

1. 安装页面会自动显示「网络加速选项」面板
2. 根据当前步骤，选择对应的加速镜像：
   - **Docker 拉取慢** → 选择 1Panel/DaoCloud/阿里云镜像
   - **git clone 慢** → 选择 GitHub 加速代理
   - **npm install 慢** → 选择淘宝/腾讯 npm 镜像
3. 点击后会**自动切换镜像并重试当前步骤**（不会重启整个安装）

> 💡 **可以去喝杯水再回来**，安装慢是正常的，不影响最终结果。

**安装失败怎么办？**
- 页面会显示错误信息
- 可以使用上方的镜像加速切换源后重试
- 或者点击「重试安装」重新开始

### 第六步：安装完成

安装完成后，页面会自动跳转到**管理面板仪表盘**。

你会看到：
- 三个服务的运行状态（NapCat、AstrBot、SillyTavern）
- 系统信息（IP、内存、磁盘等）
- **云酒馆登录信息**（用户名和初始密码）
- **NapCat Token**（用于 NapCat 登录）
- 配置教程链接

> ⚠️ **重要：请立即记录云酒馆的初始密码！** 仪表盘上会显示用户名（`default-user`）和初始密码。

## ❓ 常见问题

### 打不开管理面板页面？

1. **检查端口**：确认你在云服务商控制台开放了 9999 端口
2. **检查 IP**：确认你输入的是公网 IP，不是内网 IP
3. **检查脚本**：回到终端，看脚本是否报错了
4. **检查防火墙**：有些服务器默认开启了 ufw 防火墙

### 安装脚本报错 "请使用 root 用户运行"？

在终端输入以下命令切换到 root 用户：
```bash
sudo su
```
然后重新运行安装命令。

### SillyTavern 打不开？

1. 在管理面板检查 SillyTavern 服务状态是否为「运行中」
2. 如果显示「已停止」，点击「重启」按钮
3. 确认你在云服务商控制台开放了 SillyTavern 的端口

### Docker 镜像拉取一直失败？

国内服务器可能无法直接访问 Docker Hub，解决方法：
1. 在安装页面的镜像加速面板中切换镜像源
2. 推荐使用「1Panel 镜像」或「DaoCloud 镜像」

### npm install 一直卡住或报错？

1. 使用安装页面的镜像加速面板切换 npm 源
2. 推荐使用「淘宝 npm 镜像」
3. npm install 有时会显示警告（warning）但实际已安装成功，程序会自动检测

### 安装完后怎么再次打开管理面板？

如果管理面板关闭了，在终端运行：
```bash
bash /opt/yolushiki/install.sh
```

如果已经安装完成，管理面板会自动注册为常驻服务，重启服务器后也会自动启动。
